## Core Philosophy

From GDD Section 10.1:

**"Pay for TIME, not POWER"**

```
┌─────────────────────────────────────────────────────────────────┐
│                     ACCEPTABLE                                   │
├─────────────────────────────────────────────────────────────────┤
│  • Speed up construction (time savings)                         │
│  • Additional builders (parallel progress)                      │
│  • Cosmetic items (self-expression)                             │

│  • VIP subscription (convenience features)                      │
│  • Resource packs (skip grinding, not exclusive)                │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    NEVER ACCEPTABLE                              │
├─────────────────────────────────────────────────────────────────┤

│  • Stats only available for purchase                            │

│  • Gambling mechanics of any kind                               │
│  • Pressure tactics ("Buy now or lose!")                        │
└─────────────────────────────────────────────────────────────────┘
```

## Revenue Streams

### 1. Premium Currency (Gems)

From GDD Section 10.2.1:

| Package | Gems | Robux | $/Gem | Bonus |
|---------|------|-------|-------|-------|
| Handful | 80 | 99 | 1.24 | Base |
| Pouch | 500 | 449 | 0.90 | +11% |
| Bag | 1,200 | 899 | 0.75 | +25% |
| Box | 2,500 | 1,599 | 0.64 | +35% |
| Chest | 6,500 | 3,399 | 0.52 | +50% |
| Vault | 14,000 | 5,999 | 0.43 | +75% |

**Strategy:**
- Smallest package = entry point (low risk)
- Larger packages = better value (reward commitment)
- Middle tier most popular (anchor effect)

### 2. Battle Pass

From GDD Section 10.3:

| Aspect | Value |
|--------|-------|
| Season Length | 28 days |
| Total Tiers | 50 |
| Premium Cost | 450 Robux |
| Expected Value | 2-3x cost in items |

**Psychology:**
- Free track keeps non-payers engaged
- Premium track visible to all (aspiration)
- Can complete through play, not just pay
- Final tier is marquee reward

### 3. VIP Subscription

From GDD Section 10.4:

| Benefit | Value |
|---------|-------|
| 50 Gems daily | 1,500/month |
| 2nd Builder permanent | 250 gem value |
| +20% resource production | Significant |
| +10% battle loot | Extra resources |
| -20% upgrade times | Faster progress |
| VIP badge | Status |

**Monthly Cost:** 450 Robux
**Value Proposition:** Daily value > one-time purchase

### 4. Starter Packs (One-Time)

From GDD Section 10.5:

| Pack | Robux | Contents | Unlock |
|------|-------|----------|--------|
| Beginner's | 99 | 500 gems, 10K resources, Rare Troop | TH1-2 |
| Builder's | 249 | 2nd Builder, 1K gems, speedups | TH3+ |
| Commander's | 499 | Rare Commander, 2.5K gems, 50K resources | TH5+ |
| Conqueror's | 999 | Epic Commander, 6K gems, speedups | TH7+ |
| Legend | 1,999 | Legendary Commander, 15K gems, Skin | TH9+ |

**Strategy:**
- Low first purchase (99 Robux) converts free players
- Gated by progression (relevant when unlocked)
- Exceptional value (better than gem equivalent)

## Pricing Strategy

### A/B Testing Plan
```
TEST 1: Starter Pack Price
- Control: 99 Robux
- Test A: 79 Robux
- Test B: 129 Robux
- Measure: Conversion rate, revenue

TEST 2: Battle Pass Price
- Control: 450 Robux
- Test A: 400 Robux
- Test B: 500 Robux
- Measure: Purchase rate, completion rate

TEST 3: Gem Package Sizes
- Control: Current sizes
- Test: Larger sizes with better bonuses
- Measure: ARPPU
```

### Regional Considerations
- Roblox handles currency conversion
- Monitor conversion rates by region
- Adjust offers based on purchasing power
